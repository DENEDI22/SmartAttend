Replace the Phase 1 stub in `dummy_client/main.py` with the full MQTT client implementation. Use paho-mqtt v2 API (`CallbackAPIVersion.VERSION2`) to match the server pattern from `app/services/mqtt.py`.

The complete implementation:

**Environment config (module level):**
- `DEVICE_ID = os.environ.get("DEVICE_ID", "unknown")`
- `ROOM = os.environ.get("ROOM", "unknown")`
- `MQTT_BROKER = os.environ.get("MQTT_BROKER", "localhost")`
- `MQTT_PORT = int(os.environ.get("MQTT_PORT", "1883"))`
- `LUX_VALUE = float(os.environ.get("LUX_VALUE", "400"))`

**Logging:** Use `logging.basicConfig` with format `%(asctime)s [{DEVICE_ID}] %(levelname)s %(message)s`, level INFO.

**Module-level `stop_event = threading.Event()`** for graceful shutdown.

**Callbacks (v2 signatures):**
- `on_connect(client, userdata, connect_flags, reason_code, properties)`: If reason_code == 0, publish registration JSON `{"device_id": DEVICE_ID}` to `devices/register` (qos=0), subscribe to `attendance/device/{DEVICE_ID}` (qos=1), log both. If reason_code != 0, log error.
- `on_message(client, userdata, message)`: Decode payload as UTF-8, log at INFO level: `"TOKEN URL RECEIVED: {url}"`.

**Periodic loops (run in daemon threads):**
- `heartbeat_loop(client)`: While not stop_event.is_set(), if client.is_connected() publish `"online"` to `devices/{DEVICE_ID}/status` (qos=0), then `stop_event.wait(30)`.
- `lux_loop(client)`: While not stop_event.is_set(), if client.is_connected() publish `str(LUX_VALUE)` to `sensors/{DEVICE_ID}/lux` (qos=0), then `stop_event.wait(60)`.

**main() function:**
1. Log startup with DEVICE_ID, ROOM, LUX_VALUE
2. Create `mqtt.Client(callback_api_version=mqtt.CallbackAPIVersion.VERSION2, client_id=f"dummy-{DEVICE_ID}")`
3. Set on_connect, on_message callbacks
4. Call `client.reconnect_delay_set(min_delay=1, max_delay=30)`
5. Register SIGTERM and SIGINT handlers that call `stop_event.set()`
6. Start heartbeat_loop and lux_loop as daemon threads
7. Try `client.connect(MQTT_BROKER, MQTT_PORT, keepalive=60)` with try/except for ConnectionRefusedError/OSError (log warning, broker will retry)
8. Call `client.loop_start()`
9. Block on `stop_event.wait()`
10. On unblock: `client.loop_stop()`, `client.disconnect()`, log shutdown

**`if __name__ == "__main__": main()`** guard.

Ensure all functions (on_connect, on_message, heartbeat_loop, lux_loop) are importable at module level for testing.
