  <action>
    Create `app/routers/student.py` with a single GET endpoint at `/student` (per D-01, D-02, D-03, D-04, D-05).

    The router:
    - `router = APIRouter(prefix="/student", tags=["student"])`
    - Single endpoint `@router.get("")` with `user: User = Depends(require_role("student"))` and `db: Session = Depends(get_db)`

    Query logic (STUD-01 + STUD-02):
    1. Compute `cutoff_date = date.today() - timedelta(days=365)` (last 12 months per D-04)
    2. Get global late threshold: `global_threshold = int(SystemSetting.get_value(db, "late_threshold_minutes", "10"))`
    3. Query all AttendanceTokens where `lesson_date >= cutoff_date`, joined with ScheduleEntry where `ScheduleEntry.class_name == user.class_name`. Order by `lesson_date DESC, start_time DESC`.
    4. For each token+entry pair, determine if student attended:
       - Get all token IDs for same schedule_entry_id + lesson_date (token rotation)
       - Check if AttendanceRecord exists for this student with any of those token IDs
       - If record exists: compute late_cutoff = `datetime.combine(token.lesson_date, entry.start_time) + timedelta(minutes=entry.late_threshold_minutes if entry.late_threshold_minutes is not None else global_threshold)`. If `record.checked_in_at > late_cutoff` then status="Verspaetet", else status="Anwesend"
       - If no record: status="Abwesend"
       - Get room from Device via entry.device_id
    5. Deduplicate: group by (schedule_entry_id, lesson_date) to avoid counting rotated tokens as separate lessons
    6. Build lesson list: each item has `date` (lesson_date formatted as "DD.MM.YYYY"), `class_name`, `start_time` (HH:MM), `end_time` (HH:MM), `room`, `status`
    7. Compute summary stats (STUD-01 per D-02, D-03): `total` = len(lessons), `attended` = count where status=="Anwesend", `late` = count where status=="Verspaetet", `missed` = count where status=="Abwesend", `percentage` = round((attended + late) / total * 100) if total > 0 else 0
    8. Group lessons by month for template (D-05): use `itertools.groupby` on lessons (already sorted by date DESC). Group key = `lesson_date.strftime("%B %Y")` with German month names. Use a MONTH_NAMES dict: `{1: "Januar", 2: "Februar", 3: "Maerz", 4: "April", 5: "Mai", 6: "Juni", 7: "Juli", 8: "August", 9: "September", 10: "Oktober", 11: "November", 12: "Dezember"}`. Group key format: `f"{MONTH_NAMES[lesson_date.month]} {lesson_date.year}"`.
    9. Pass to template: `stats` dict with keys total/attended/missed/late/percentage, `groups` list of dicts each with `month` (string) and `lessons` (list)

    In `app/main.py`, add after the existing teacher_router import:
    ```python
    from app.routers.student import router as student_router
    ```
    And register: `app.include_router(student_router)` right after the teacher_router line.
