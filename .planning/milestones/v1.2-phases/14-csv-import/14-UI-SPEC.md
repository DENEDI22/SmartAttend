## Color

Pico CSS semantic variables. No custom hex values.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--pico-background-color)` | Page background |
| Secondary (30%) | `var(--pico-card-background-color)` | Tables, form sections |
| Accent (10%) | `var(--pico-primary)` | "Import bestaetigen" confirm button, template download link |
| Destructive | `var(--pico-del-color)` | Error row backgrounds in validation preview table |

Accent reserved for: confirm import button, template download button/link. All other buttons use default Pico styling.

### Error Row Styling

Invalid rows in validation preview use inline style:
```
background: color-mix(in srgb, var(--pico-del-color) 15%, transparent);
```
This provides a light red tint without obscuring text. The error column text uses `var(--pico-del-color)` at full opacity.

Valid rows use no special styling (default table row).

The "Status" column shows:
- Valid rows: "OK" in `var(--pico-ins-color)` (green)
- Invalid rows: error message text in `var(--pico-del-color)` (red)

## Interaction States

### Upload Flow (D-09: two-step form replacement)

1. **Idle state**: Upload section visible at bottom of existing admin page (below existing user/device table)
2. **File selected**: Native browser file input shows filename, submit button enabled
3. **Processing**: Form submits, server returns preview page (full page load, not AJAX)
4. **Preview - all valid**: Summary in green, confirm + back buttons visible
5. **Preview - has errors**: Summary in red, error rows highlighted, only back button visible (no confirm)
6. **Success**: Redirect to original page with success message in query param (existing pattern)
7. **Error (file level)**: Redirect to original page with error message in query param (existing pattern)

### Loading State

No special loading indicator needed. Standard browser form submission behavior (page navigation). Pico CSS applies `aria-busy` styling if needed on the submit button, but CSV parsing is fast enough that no spinner is warranted.
