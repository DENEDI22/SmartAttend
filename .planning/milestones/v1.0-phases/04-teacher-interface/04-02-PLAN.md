    **Add to `app/routers/teacher.py`:**

    Add imports at top: `import csv`, `import io`, `from fastapi.responses import Response`

    **Helper function `_build_roster(db, schedule_entry, token)`:**
    - Query all active students where `role == "student"` AND `class_name == schedule_entry.class_name` AND `is_active == True`, ordered by `last_name` then `first_name` (per Claude discretion: alphabetical by last name)
    - If token exists, query all AttendanceRecord where `token_id == token.id`, build dict `{record.student_id: record}`
    - For each student, build dict: `last_name`, `first_name`, `status` ("Anwesend" if in records else "Abwesend"), `time` (record.checked_in_at.strftime("%H:%M") if present, else "" per D-11)
    - Return roster list, checked_in count, expected count

    **GET `/teacher/lesson/{token_id}`** (per D-05, D-06, D-07):
    - Protected by `require_role("teacher")`
    - Query AttendanceToken by id (`token_id`). If not found, redirect to `/teacher?error=Stunde+nicht+gefunden.` (per D-14 flash pattern)
    - Query ScheduleEntry by `token.schedule_entry_id`. Verify `schedule_entry.teacher_id == user.id`. If mismatch, redirect to `/teacher?error=Sie+haben+keinen+Zugriff+auf+diese+Stunde.`
    - Query Device by `schedule_entry.device_id` to get room
    - Call `_build_roster(db, schedule_entry, token)` to get roster, checked_in, expected
    - Pass to template: `request`, `roster`, `class_name=schedule_entry.class_name`, `room=device.room`, `start_time=schedule_entry.start_time.strftime("%H:%M")`, `end_time=schedule_entry.end_time.strftime("%H:%M")`, `checked_in`, `expected`, `token_id`, `active_page="overview"`

    **GET `/teacher/lesson/{token_id}/csv`** (per D-08, D-09, D-10, D-11):
    - Protected by `require_role("teacher")`
    - Same token lookup + teacher ownership check as lesson detail (redirect on error)
    - Call `_build_roster(db, schedule_entry, token)` to get roster
    - Create `io.StringIO()`, write UTF-8 BOM `"\ufeff"` first
    - Create `csv.writer(output, delimiter=";")` per D-08
    - Write header row: `["Nachname", "Vorname", "Klasse", "Status", "Uhrzeit"]`
    - Write each roster entry: `[row["last_name"], row["first_name"], schedule_entry.class_name, row["status"], row["time"]]`
    - Filename: `Anwesenheit_{schedule_entry.class_name}_{token.lesson_date.strftime('%Y-%m-%d')}.csv` per D-10
    - Return `Response(content=output.getvalue(), media_type="text/csv; charset=utf-8", headers={"Content-Disposition": f'attachment; filename="{filename}"'})`

    **Create `app/templates/teacher_lesson.html`** (per D-05, D-06, D-07):
    - Extends `teacher_base.html`
    - Title block: `Anwesenheit {{ class_name }} -- SmartAttend`
    - Back link: `<a href="/teacher">&larr; Zurueck zur Uebersicht</a>`
    - Header: `<hgroup><h1>Anwesenheit {{ class_name }}</h1><p>{{ room }} -- {{ start_time }} bis {{ end_time }} -- Anwesend: {{ checked_in }}/{{ expected }}</p></hgroup>` (per D-06)
    - Table: `<figure><table>` with headers: Nachname, Vorname, Status, Uhrzeit
    - Each row: `row.last_name`, `row.first_name`, status with color (`style="color: #2e7d32"` for Anwesend, `style="color: var(--pico-muted-color)"` for Abwesend), `row.time` or `--` if empty
    - Below table: `<a href="/teacher/lesson/{{ token_id }}/csv" role="button" class="outline">CSV herunterladen</a>` (per D-07)
