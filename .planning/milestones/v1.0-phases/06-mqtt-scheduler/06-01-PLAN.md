    Module-level `_client: mqtt.Client | None = None`

    `_on_connect(client, userdata, connect_flags, reason_code, properties)`:
    - If reason_code == 0: subscribe to [("devices/register", 0), ("devices/+/status", 0), ("sensors/+/lux", 0)] (per D-02: QoS 0 for these)
    - Log connection success or failure (per D-12)

    `_on_message(client, userdata, message)`:
    - Decode payload as UTF-8
    - Route by topic: "devices/register" -> _handle_register, devices/X/status -> _handle_heartbeat(X), sensors/X/lux -> _handle_lux(X)
    - Wrap in try/except with logger.exception (per D-12)

    `_handle_register(payload: str)`:
    - Parse JSON: `json.loads(payload)` -> extract `device_id` key (per D-01: JSON format)
    - Fallback: if JSON parse fails, use `payload.strip()` as device_id
    - Create own `SessionLocal()` session (per D-11)
    - Check if device exists (query by device_id); if yes, return (idempotent)
    - Create Device(device_id=device_id, is_enabled=False, is_online=False)
    - Commit, log registration (per D-12: log new registrations)
    - Finally: db.close()

    `_handle_heartbeat(device_id: str, payload: str)`:
    - Create own `SessionLocal()` session (per D-11)
    - Query Device by device_id; if not found, return silently
    - Set device.is_online = True, device.last_seen = datetime.now()
    - Commit. Finally: db.close()
    - No logging for routine heartbeats (per D-12)

    `_handle_lux(device_id: str, payload: str)`:
    - Create own `SessionLocal()` session (per D-11)
    - Query Device by device_id; if not found, return silently
    - Set device.last_lux = float(payload)
    - Commit. Finally: db.close()
    - No logging for routine lux (per D-12)

    `publish_token(device_id: str, url: str)`:
    - If _client and _client.is_connected(): publish to f"attendance/device/{device_id}" with payload=url, qos=1 (per D-02: QoS 1 for token delivery)

    `start_mqtt()`:
    - global _client
    - settings = get_settings()
    - _client = mqtt.Client(callback_api_version=mqtt.CallbackAPIVersion.VERSION2)
    - Set on_connect = _on_connect, on_message = _on_message
    - _client.reconnect_delay_set(min_delay=1, max_delay=30)
    - _client.connect(settings.mqtt_broker, settings.mqtt_port, keepalive=60)
    - _client.loop_start() (per D-10: threaded background loop)

    `stop_mqtt()`:
    - global _client
    - If _client: loop_stop(), disconnect(), set to None

    Use `import logging; logger = logging.getLogger(__name__)` for all logging.
