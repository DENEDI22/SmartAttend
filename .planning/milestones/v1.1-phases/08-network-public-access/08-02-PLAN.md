   ```yaml

   # client-dev:

   #   environment:

   #   restart: unless-stopped
   ```

3. **Add** the ngrok service after the mqtt service (per D-06, D-07, D-08):
   ```yaml
   ngrok:
     image: ngrok/ngrok:latest
     command: http server:8000 --domain=${NGROK_DOMAIN}
     environment:
       - NGROK_AUTHTOKEN=${NGROK_AUTHTOKEN}
     ports:
       - "4040:4040"
     depends_on:
       - server
     restart: unless-stopped
   ```
   Notes:
   - `command: http server:8000 --domain=${NGROK_DOMAIN}` tunnels to server container on port 8000 using the reserved domain
   - `NGROK_AUTHTOKEN` is passed via environment (ngrok reads it automatically)
   - Port 4040 exposes the ngrok web inspection interface (useful for debugging)
   - The `${NGROK_DOMAIN}` and `${NGROK_AUTHTOKEN}` are interpolated from `.env` by Docker Compose

4. **Keep** the existing `server` and `mqtt` services unchanged. Keep `volumes:` section with `mosquitto-data:`.
